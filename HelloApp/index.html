iOS Wi-Fi Proxy Profile
Профиль работает только когда телефон подключён к указанной Wi-Fi сети. На мобильном интернете и других сетях прокси не действует — это ограничение iOS, не баг.
Wi-Fi сеть (SSID)
Прокси сервер
Как поставить профиль:
  1. Заполни SSID, хост и порт, тапни "Создать профиль"
  2. Откроется окно "Поделиться" — выбери "Сохранить в Файлы" (или сразу открой)
  3. Настройки → сверху появится "Профиль загружен"
  4. Установить → введи код-пароль → Установить
  5. Прокси активен только когда ты на этой Wi-Fi сети
Чтобы выключить — жми "Stop" выше и установи получившийся профиль тем же способом. Он перезапишет текущий и снимет прокси.
